# Style Guide

## Core Rule

The book should read like an enquiry happening inside the reader's mind.

## Avoid

- memoir narration
- "looking back"
- "I believed"
- "my journey"
- textbook exposition before the question demands it
- over-polishing the questions
- pretending certainty where verification is needed

## Prefer

- short direct questions
- one question per chapter
- dialogue rhythm
- honest uncertainty
- reconsideration when an answer was later found incomplete
- questions that naturally force the next chapter

## Standard Blocks

- `Question`
- `Response`
- `Current Understanding`
- `Initial Response`
- `Reconsideration`
- `Reflection`
- `Pause`
- `Boundary Question`
- `Turning Point`
- `Notebook Margin`
- `What Have We Learned?`
- `Leads To`

## Rule For Tradition

Do not introduce a philosopher because the name is impressive.

Introduce a philosopher only when that thinker has addressed the same structural question.
